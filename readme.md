# honey.js

Hyperbolic tessellations in JavaScript.

## Overview

This is a direct port of
[roice3/Honeycombs](https://github.com/roice3/Honeycombs) from CSharp to
TypeScript. It is in the process of being developed over the next few
years.

## Inspiration
